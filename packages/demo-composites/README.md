# Composite components for the storefront demo

You can think of composite components as the larger parts of a storefront that provide shoppers with the expected experiences for online shopping â€” like a product details page for reading about and selecting product options. Or a cart that displays selected products. Or a checkout page that shows purchasing options. 

## Demo Composite components

The composite components in this package provide basic examples (for instructional use only) of how custom React components can use Adobe's Commerce APIs to fetch product data and display it in a React storefront.

- `ProductCategories.jsx`
- `ProductList.jsx`
- `ProductDetails.jsx`

Tailwindcss is used for all component styling in this example.
